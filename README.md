# SIP & Lump Sum Calculator

An interactive investment calculator built using HTML, CSS, and JavaScript.

## Features

- SIP Calculator
- Lump Sum Calculator
- Dynamic Investment Growth Chart
- Canvas API Visualization
- Responsive Design
- Indian Currency Formatting

## Tech Stack

- HTML
- CSS
- JavaScript
- Canvas API

## Formulae Used

### SIP

FV = P × [((1 + r)^n - 1) / r] × (1 + r)

### Lump Sum

FV = P × (1 + r)^n
